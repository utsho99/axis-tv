# AXIS TV

A premium IPTV streaming interface built with React, TypeScript, and Vite.

![License](https://img.shields.io/badge/license-GPLv3-DC143C?style=flat-square)
![React](https://img.shields.io/badge/React-18-61DAFB?style=flat-square&logo=react)
![TypeScript](https://img.shields.io/badge/TypeScript-5-3178C6?style=flat-square&logo=typescript)

## Features
- Live IPTV channel streaming via HLS
- Channel browsing, favorites, and recent history
- Crimson glassmorphic UI
- Search, settings, and M3U import

## Tech Stack
- React 18 + TypeScript
- Vite + Tailwind CSS
- Framer Motion
- HLS.js

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
